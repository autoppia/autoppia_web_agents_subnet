# Docker Installation

Script to automatically install Docker and Docker Compose.

## Requirements:

- Ubuntu/Debian
- Sudo privileges

## How to use:

1. Make it executable:

```bash
chmod +x install_docker.sh
```

2. Run it:

```bash
./install_docker.sh
```

3. Log out and log back in

## What it installs:

- Docker
- Docker Compose v2
- User permissions
